💠 Installation Basics

First, go to Arch Linux Download Page and scroll down to find your country. Click on whatever link you want, and then download the ISO image you want to download (generally the top/latest one with the release date). Now create a bootable USB drive using Rufus. For partition scheme use GPT, and for File system, use fat32 with default cluster size.

After that, reboot your computer with your usb stick inserted and enter in the BIOS. Set the boot order to boot from your USB drive. Save and exit BIOS

When the computer boot up, select the default option. For simplicity, use an ethernet cable, but if you can't, you will need to install your wifi driver, follow this part of the guide and come back.

Now, you need your keyboard layout to fit your needs. So, if you have a French Azerty keyboard, you will need to run this command: 

loadkeys fr

Now, for an easy setup, you will use the following command : 

archinstall

🌀 Discover Hyprland

Select Hyprland (not hyprland-uwsm-managed), and login !

You can see the Hyprland version, when i write this, it's 0.49.0. Close the windows by clicking in the "Thanks!" button.

Press Super (= windows) + Q to open the terminal.

I will install my browser (Brave) for copy paste command line directly to the terminal. But for installing Brave, we need yay, so :

  > git clone https://aur.archlinux.org/yay.git
  > cd yay
  > makepkg -si

Now, we can install brave :

yay -S brave-bin

We can check for system-wide updates :

sudo pacman -Syu

⬇️ Full installing script


    set -e
    PKG_LIST=(zsh alacritty neovim wofi waybar swww mako thunar lf cliphist wl-clipboard ttf-iosevka-nerd noto-fonts noto-fonts-emoji chezmoi)
    sudo pacman -Syu --noconfirm && sudo pacman -S --noconfirm "${PKG_LIST[@]}"
    chsh -s /bin/zsh
    sh -c "$(curl -fsSL https://raw.githubusercontent.com/zdharma-continuum/zinit/HEAD/scripts/install.sh)"
    command -v yay &>/dev/null || (git clone https://aur.archlinux.org/yay.git && cd yay && makepkg -si --noconfirm && cd .. && rm -rf yay)
    yay -S --noconfirm swaylock-effects papirus-icon-theme papirus-folders
    git clone https://github.com/catppuccin/papirus-folders.git && cd papirus-folders && sudo cp -r src/* /usr/share/icons/Papirus/ && cd .. && rm -rf papirus-folders
    papirus-folders -C cat-mocha-blue --theme Papirus
    ALL_PKGS=("${PKG_LIST[@]}" swaylock-effects papirus-icon-theme papirus-folders)
    NOT_FOUND=(); for pkg in "${ALL_PKGS[@]}"; do pacman -Q "$pkg" &>/dev/null || yay -Q "$pkg" &>/dev/null || NOT_FOUND+=("$pkg"); done
    [ ${#NOT_FOUND[@]} -eq 0 ] && echo "✅ Tous les paquets sont installés." || { echo "❌ Manquants :"; for p in "${NOT_FOUND[@]}"; do echo "  - $p"; done; exit 1; }
